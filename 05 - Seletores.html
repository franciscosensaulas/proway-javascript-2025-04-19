Exemplo de Seletores

Seletor de id

O método document.getElementById é utilizado em JavaScript para acessar e manipular um elemento HTML com base no seu atributo id. Ele retorna uma referência ao primeiro elemento que possui o id especificado, permitindo que você altere seu conteúdo, estilo, ou adicione eventos de forma dinâmica. É uma das formas mais diretas e comuns de interagir com o DOM (Document Object Model).

Seletor de tag

O método document.getElementsByTagName é usado em JavaScript para obter uma coleção de todos os elementos HTML com a tag especificada, como 'div', 'p', ou 'li'. Ele retorna uma coleção HTML (HTMLCollection) com todos os elementos correspondentes, permitindo que você acesse e manipule vários elementos ao mesmo tempo usando índices ou loops. Esse método é útil para aplicar alterações em grupo a elementos do mesmo tipo dentro da página.
