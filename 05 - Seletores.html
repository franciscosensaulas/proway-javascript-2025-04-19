Exemplo de Seletores

Seletor de id

O método document.getElementById é utilizado em JavaScript para acessar e manipular um elemento HTML com base no seu atributo id. Ele retorna uma referência ao primeiro elemento que possui o id especificado, permitindo que você altere seu conteúdo, estilo, ou adicione eventos de forma dinâmica. É uma das formas mais diretas e comuns de interagir com o DOM (Document Object Model).

Seletor de tag

O método document.getElementsByTagName é usado em JavaScript para obter uma coleção de todos os elementos HTML com a tag especificada, como 'div', 'p', ou 'li'. Ele retorna uma coleção HTML (HTMLCollection) com todos os elementos correspondentes, permitindo que você acesse e manipule vários elementos ao mesmo tempo usando índices ou loops. Esse método é útil para aplicar alterações em grupo a elementos do mesmo tipo dentro da página.

Seletor de name

O método document.getElementsByName é utilizado em JavaScript para selecionar todos os elementos HTML que possuem um atributo name com o valor especificado. Ele retorna uma NodeList (semelhante a um array) contendo todos os elementos correspondentes, o que é especialmente útil em formulários para acessar inputs, radio buttons e checkboxes com o mesmo nome. Diferente de getElementById, ele pode retornar múltiplos elementos.

Seletor de class name

O método document.getElementsByClassName é utilizado em JavaScript para selecionar todos os elementos HTML que possuem uma determinada classe CSS. Ele retorna uma HTMLCollection com todos os elementos que compartilham essa classe, permitindo manipulações em grupo, como alteração de estilo, conteúdo ou eventos. É muito útil para aplicar mudanças consistentes a vários elementos com a mesma aparência ou função.
